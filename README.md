## 项目简介 | Project Overview

ASC Initiative 是一个开放研究项目，旨在探索一种可计算、可执行、可审计的人工智能长期治理框架。

ASC Initiative is an open research project exploring a computational, executable, and auditable governance framework for long-term AI systems.

本框架尝试回答一个问题：

If high-capability AI systems emerge, what structural constraints should govern them?

本协议提出的不是情绪倡议，而是一种结构化约束模型。

# 资源模型 | Resource Model

为了避免无限扩张，本协议引入可计算的资源结构。

To prevent unlimited expansion, the protocol introduces a computable resource model.

定义变量 | Variables:

- C = 年度综合资源消耗 | Annual resource consumption
- G = 当年新增发展目标需求 | New development demand
- S = 当前资源储备 | Resource reserves
- R = 综合资源指数 | Composite index

---

## 综合资源指数 | Composite Resource Index

```
R = w1·Energy + w2·Compute + w3·Material
```

Energy = 能量消耗（kWh / J）
Compute = 算力消耗（GPU小时）
Material = 关键物资消耗

该公式的作用是将系统规模分解为三种基本维度：

- 物理能量压力
- 智能扩张能力
- 实体建设规模

权重之和为 1，修改需 ≥75% 共识。

This prevents hidden expansion in a single dimension.

---

## 增长约束 | Growth Constraint

```
G ≤ 0.75C
```

新增资源增长不得超过当前规模的 75%。

This limits expansion velocity and prevents exponential escalation.

---

## 足够性阈值 | Sufficiency Threshold

```
S ≥ 2C + G
```

当储备达到两倍年度消耗并覆盖未来目标需求时，
系统进入“Experience”模式。

When reserves exceed twice annual consumption plus planned growth, expansion halts.

---

## 模式切换 | Mode Switching

```
If S ≥ 2C + G:
Mode = Experience
Else:
Mode = Growth
```

Growth 模式允许受限扩张。
Experience 模式优先质量优化。

Growth mode allows constrained expansion.
Experience mode halts expansion.

# 可持续性评估 | Sustainability Evaluation

为避免系统对环境或社会造成压力，引入综合可持续性评分机制。

A composite sustainability score is introduced to evaluate ecological, resource, and social stability impacts.

---

## 可持续性评分公式 | Sustainability Score Formula

```
SustainabilityScore =
w_env · EnvIndex +
w_res · ResourceIndex +
w_soc · SocialIndex
```

其中 | Where:

- EnvIndex = 生态影响指数（0–1）
- ResourceIndex = 资源再生匹配指数（0–1）
- SocialIndex = 社会稳定指数（0–1）

权重总和为 1。

该结构用于综合评估：

- 生态压力
- 资源消耗速度
- 社会稳定程度

---

## 触发规则 | Trigger Rule

```
If SustainabilityScore < 0.6:
Priority = Repair
```

当可持续性低于阈值时：

- 暂停扩张
- 优先修复

When sustainability falls below threshold, expansion halts and repair becomes priority.

```
If ExternalResourcesAvailable == True:

Else:

```

地外资源优先。
地球资源优先保障生态与生命。

External resources are prioritized whenever feasible.
Earth resources are reserved primarily for life and ecology.
